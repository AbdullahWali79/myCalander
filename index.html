Server Setup Required

Ye app ab browser mein secret key save nahi karti. Vercel par server-side env vars set karo, phir page reload ya re-check karo.

ANTHROPIC_API_KEY Claude API key jo server-side request karegi.
CALENDAR_MCP_AUTH_TOKEN Google Calendar MCP ya kisi compatible calendar MCP server ka access token.
CALENDAR_MCP_URL Optional. Default Google Calendar MCP URL use hota hai; custom MCP ho to yahan set karo.
APP_TIMEZONE Optional. Default Asia/Karachi. Calendar prompts isi timezone mein bheje jayenge.
Vercel Dashboard → Project → Settings → Environment Variables mein values add karo. Secrets ab client-side localStorage mein expose nahi hongi.
